`py32-metapac` is an all-in-one Peripheral Access Crate (PAC) for Puya PY32 MCU families.

This crate is generated from [py32-data](https://github.com/py32-rs/py32-data).  
Please do not edit this crate directly.

https://github.com/py32-rs/py32-hal provides a cross-family HAL implementation depending on py32-metapac.

API documentation is available on docs.rs: <https://docs.rs/py32-metapac>.
